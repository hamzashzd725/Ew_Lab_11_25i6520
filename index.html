// 01 — the guide

RICING
FUNDAMENTALS

Follow these steps to transform a vanilla Linux install into a personalized, refined setup. No shortcuts, no prebuilt themes — do it yourself.

01
Choose a Distro

Start with a distro that gets out of your way. Arch gives you a blank canvas. NixOS gives you reproducibility. Pick your poison.

Arch NixOS Gentoo Void
02
Pick a Window Manager

Ditch the DE. A tiling WM like bspwm, i3, or Hyprland gives you full keyboard-driven control and a blank slate for aesthetics.

bspwm i3 Hyprland dwm
03
Configure Your Terminal

Your terminal is your primary interface. Choose a fast emulator (kitty, Alacritty), configure zsh with plugins, and pick a colorscheme.

kitty Alacritty zsh tmux
04
Set a Colorscheme

A consistent palette ties your entire setup together. Apply it across your WM borders, terminal, bar, nvim, rofi, and GTK apps.

Catppuccin Gruvbox Dracula Nord
05
Build a Status Bar

Polybar, Waybar, or eww for a fully custom bar. Show workspaces, system stats, time, and whatever modules matter to you.

Polybar Waybar eww
06
Version Your Dotfiles

Track everything with Git. Use GNU Stow to symlink your config files. Push to GitHub so your setup is reproducible anywhere.

git stow chezmoi
// 02 — toolbox

ESSENTIAL
TOOLS

Curated list of tools used by the ricing community. Mix and match — there's no single right setup.

Compositor picom X11 compositor for transparency, shadows, blur, and animation effects.
Compositor (Wayland) Hyprland Dynamic tiling Wayland compositor with buttery animations and extensive config.
App Launcher rofi Highly customizable window switcher and application launcher.
Notifications dunst Lightweight, customizable notification daemon with full theme support.
Wallpaper feh / swww feh for X11, swww for Wayland — set and rotate wallpapers with transitions.
Fetch fastfetch Blazing-fast system info display. neofetch successor, fully configurable.
File Manager lf / yazi Terminal-based file managers with vim keybindings and image preview.
Editor Neovim Extensible modal editor. Theming with lazy.nvim and colorscheme plugins.
Shell Prompt starship Cross-shell prompt written in Rust. Minimal, informative, and fast.

EXAMPLES

~/.config/kitty/kitty.conf config
# Font font_family JetBrains Mono font_size 11.5 # Padding & transparency window_padding_width 20 background_opacity 0.92 # Catppuccin Mocha foreground #cdd6f4 background #1e1e2e color0 #45475a color1 #f38ba8 color2 #a6e3a1 color4 #89b4fa # Misc cursor_shape beam enable_audio_bell no
